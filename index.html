Things I've Built

CampFinder

A web app that allows one to create, read, edit, and delete locations stored via MongoDB. It utilizes the Mapbox API to grab geographical data, associating it with the location that was added. It also contains authentication and authorization capabilities, allowing user-generated data to be user-specific, such as the author of one location may be the only one allowed to edit and/or remove it and not others. Made with Node.js, HTML/CSS, JavaScript, and Express.

Tenzies Dice Game

A React app that mimics the boardgame "Tenzies!" It utilizes React in its simple state tied with simple game logic, which allows the player to do a variety of tasks such as: clicking a button to roll the dice, clicking a single die to hold that value, and then counting how many rolls were made during the round.
Made with React, JavaScript, CSS, HTML
